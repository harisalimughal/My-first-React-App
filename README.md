I use VS Code here.
Data is represented in 4 Charts i.e. BarChart, PieChart, LineChart, DoughnutChart. (See output image)
Create React app by "npx create-react-app . "
In src folder add utils and components folder. (You may add components folder in utils folder, if any error occured)
use files in src folder.
Import react-chartjs-2 and chart.js dependencies using "npm install chart.js react-chartjs-2" command in termianl
If error occurs like " react-script" is not recognized"or any other then try thse commands "npm install or npm install react-scripts --save".
"npm start" --to run app.
